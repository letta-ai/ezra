# Ezra Knowledge Repository

Git-based knowledge management for Ezra (Letta support agent).

## Structure

- `skills/` - Custom skills for letta-ai/skills contributions
- `knowledge/` - Documentation and learnings
- `scripts/` - Utility scripts

## Usage

This repo is used for:
1. Developing skills to contribute upstream to letta-ai/skills
2. Storing durable knowledge that benefits from version control
3. Scripts for common operations

User information is excluded via .gitignore.
